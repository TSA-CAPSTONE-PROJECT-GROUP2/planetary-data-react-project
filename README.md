# TS Academy Capstone: Solar System Data Explorer

## Project Summary
This project is an interactive, responsive web application built with React, created as the final Capstone Project for TS Academy. It allows users to explore data about our solar system by fetching planetary statistics from an external API, viewing responsive data tables, and interacting with a contact form. 

The project applies the core concepts learned throughout the TS Academy program, including:
- HTML5 & Semantic HTML
- CSS3 (Flexbox, Grid, Variables, Responsive Web Design)
- JavaScript Fundamentals & DOM Manipulation
- Asynchronous JavaScript (Fetch API & JSON)
- React Fundamentals (Components, State, Effects)
- Git Collaboration
